Development: `yarn dev`, `yarn build`, `yarn preview`.
Quality: `yarn typecheck`, `yarn lint`, `yarn eslint`, `yarn stylelint`, `yarn prettier`, `yarn prettier:write`.
Tests: `yarn vitest`, `yarn vitest:watch`, `yarn vitest:storybook`, `yarn test`, `yarn e2e`, `yarn e2e:local:mock`, `yarn e2e:docker:real`.
Storybook: `yarn storybook`, `yarn storybook:build`.
Docker stack (from app): `docker-compose up -d`, `docker-compose logs -f`, `docker-compose down`.
Common shell utilities on Darwin: `git`, `ls`, `cd`, `rg`, `find`, `cat`, `sed`.
